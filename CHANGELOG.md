# 20HA META-X 버전 기록

본서버(meta-x.ai.kr) 배포마다 한 줄씩 추가합니다.
버전 포맷: `v{YYYY.MM.DD}-{N}` (같은 날 N번째 배포)

각 버전은 git tag로도 남아 있어 `git checkout <tag>` 로 코드 상태 복원 가능, Vercel 대시보드에서는 클릭 한 번으로 production 롤백 가능.
